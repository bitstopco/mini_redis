    Overview

    A Redis value. It's #raw value has a type according to Redis Protocol Specification:

    # Redis type    | First byte | Crystal type |
# ------------- | ---------- | ------------ |
# Simple String | `+`        | `String`     |
# Integer       | `:`        | `Int64`      |
# Bulk String   | `$`        | `Bytes`      |
# Array         | `*`        | `Array`      |
    response = String.new(redis.command("GET foo").raw.as(Bytes))
pp response # => "bar"
