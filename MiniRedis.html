    def pipeline(read : Bool, &block : Pipeline -> ) : Array(Value)? #

    Yield Pipeline, accumulate requests and then flush them all in one moment. read argument defines whether to read the response or not (improves performance, may be used in throw-away clients).

    See Pipelining docs.

  pipe.send("PING")

    def receive(*, skip_queued = false) : Value #

    Read a response. Blocks until read. The response is cast to a Crystal type according to the Redis Protocol Specification. For more information on types, see Value.

    In case of error (- byte), a Error is raised.

    There is an optional skip_queue argument, which is used in Transaction mode -- it skips QUEUED strings, which improves the performance.

    def send(command : String) : Nil #

    Send a String inline command. It's generally faster than the Enumerable version, because it skips marshalling. However, inline commands could be a worse fit for operations with binary keys or values.

    See Redis Protocol: Inline Commands for more information.
